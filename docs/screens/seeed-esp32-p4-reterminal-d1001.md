# 8-inch Seeed reTerminal D1001

The **Seeed reTerminal D1001** is a 8-inch landscape
touchscreen powered by **ESP32-P4**, with room for **20 cards**
on the home screen.

Community port contributed via [PR #885](https://github.com/jtenniswood/espcontrol/pull/885) by @zacs.

::: warning Not yet hardware-verified
This device compiles and is flashable, but nobody has confirmed it on real hardware yet. It may not work. If you own this panel, [verifying it](/reference/support-policy#hardware-verification) is the most valuable contribution you can make.
:::

## Specifications

| | |
|---|---|
| **Screen size** | 8 inches |
| **Resolution** | 1280 x 800 |
| **Orientation** | Landscape |
| **Processor** | ESP32-P4 |
| **Card grid** | 5 × 4 (20 cards) |

If you use ESPHome and prefer to compile firmware yourself:

```yaml
substitutions:
  name: "your-device-name"
  friendly_name: "Your Device Name"

wifi:
  ssid: !secret wifi_ssid
  password: !secret wifi_password

packages:
  setup:
    url: https://github.com/lamiskin/espcontrol-community-devices/
    file: devices/seeed-esp32-p4-reterminal-d1001/packages.yaml
    refresh: 1s
```
