# SenseCAP Indicator D1 Community Build

This device package is maintained by the community via the `lamiskin/espcontrol-community-devices` repository.

The SenseCAP Indicator D1 is not officially supported by upstream EspControl. Firmware builds, testing, issue triage, and user support for this device are community-provided.

Manual ESPHome package reference:

```yaml
packages:
  setup:
    url: https://github.com/lamiskin/espcontrol-community-devices/
    file: devices/seeed-sensecap-indicator-d1/packages.yaml
    refresh: 1s
```
